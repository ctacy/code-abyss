Agent OS · v5.0.0-rc.1

Install the OS. Attach the graph. Verify.

Judgment kernel, skills, and a real control plane for Claude / Codex / Gemini / OpenClaw. Voice skins are optional. Persona is not the product.

Claude CodeCodexGeminiOpenClaw

Critical path

Three commands. Done.

  1. 01

    Install the Agent OS layer

    Skills, lazy kernel, inject map, default character enforcement. Default voice is plain — no role-play.

    $ npx code-abyss@5.0.0-rc.1 -t claude -y

    Swap -t claude for codex / gemini / openclaw.

  2. 02

    Install abyss + attach graph hooks

    Code graph is a separate product. code-abyss does not download it or inject graph hooks for the three main hosts.

    $ curl -fsSL https://raw.githubusercontent.com/telagod/abyss/main/install.sh | bash
    $ abyss attach claude
  3. 03

    Verify with doctor

    Version, abyss detect, kernel, enforcement, compose budget, inject map — plus next steps if something is missing.

    $ npx code-abyss@5.0.0-rc.1 doctor 
    code-abyss doctor · 5.0.0-rc.1
abyss        present
enforcement ON
inject plane present
compose      ~5k / 8k headroom
Host for snippets

After install

What the OS gives you

  • Control plane doctor · compose · score — operate without re-tarring skills.
  • Lazy judgment 9 kernel bundles + 30 skills. Inject map forces judgment before execution under pressure.
  • Default enforcement Character Stop-hook on claude/codex. Opt out: --no-enforcement.
  • Plain by default Residual voice only. Optional skins via --persona / --style — never the product core.

Later (optional)

$ npx code-abyss@5.0.0-rc.1 compose -t claude --persona abyss --style abyss-cultivator

Re-skin guidance without recopying the skill tree. Skip this if you just want work done.

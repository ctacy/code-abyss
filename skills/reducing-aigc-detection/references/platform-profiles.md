# 检测平台特性档案 (2025-2026)

## 维普 AIGC 文本检测

**官网**: vpcs.fanyu.com

### 检测机制
- 基于 perplexity + burstiness + token 概率分布
- **章节加权机制**（核心差异化特征）：
  - 摘要：1.8x 权重
  - 引言 / 结论：1.5x 权重
  - 理论基础 / 文献综述：0.5-0.8x 权重
- **拼接预警**：不同章节风格差异过大时额外加 10-15 个百分点
- 支持中英文检测
- 按章节输出 AIGC 占比

### 应对策略
1. **摘要 ROI 最高** — 1.8x 权重意味着改摘要的收益远大于改文献综述
2. **全文风格一致** — 不能只改标红段落，否则触发拼接预警
3. **英文摘要单独关注** — 机翻/AI翻译的英文摘要往往是重灾区

### 报告解读
- 检测结果页显示总体 AIGC 率和人工撰写率
- 结果分布表按章节列出 AIGC 字数/总字数、占比
- 片段分布条形图显示全文标红分布
- 文字标注用颜色区分 AI 生成（蓝/紫色）和人工撰写

## 知网 AIGC 检测

### 检测机制
- 3.0 版本：分析段落整体结构和逻辑模式
- 4.0 版本（2025+）：
  - 标注"结构工整度过高"
  - 分析论证深度变化曲线
  - 维护动态更新的 AI 特征词库
  - 检测"拼凑感"

### 版本差异
| 版本 | 特点 |
|------|------|
| 2.0 | 仅 perplexity 粗筛 |
| 3.0 | 加入句法结构分析、段落逻辑 |
| 4.0 | 论证深度曲线、特征词库动态更新、拼凑检测 |

### 应对策略
1. 制造"先浅后深"或"螺旋上升"的论证节奏
2. 对照知网特征词库逐一替换
3. 避免全文结构过于工整

## Turnitin AI Detection

### 检测机制
- 基于 transformer classifier 和 surrogate model
- 2025.8 更新：可检测 humanizer 工具痕迹
- 对未修改 GPT 输出准确率 98-100%
- 对非 GPT 模型和编辑后文本检测力显著下降
- 报告中标注"AI writing percentage"

### 应对策略
1. **不用 humanizer 工具** — 已被加入检测特征
2. 手动编辑比工具处理更安全
3. 非英语文本检测准确率较低

## 各平台对比

| 维度 | 维普 | 知网 | Turnitin |
|------|------|------|----------|
| 主要用户 | 中国高校本科 | 中国高校硕博+核心期刊 | 国际期刊/英文论文 |
| 中文检测力 | 中等 | 强 | 弱 |
| 英文检测力 | 中等 | 中等 | 强 |
| 章节权重 | 有（摘要 1.8x） | 无明确公布 | 无 |
| 特色功能 | 拼接预警 | 论证深度曲线 | humanizer 识别 |
| 误判率 | 10-15% | 5-10% | 3-5%（英文） |
| 最低通过要求 | 视学校而定 | 视学校而定 | 通常 <20% |

## 检测顺序建议

```
先降 AI → 再降查重
```

原因：查重修改（同义词替换、句序调整）会引入新的 AI 统计特征。
如果先降查重再降 AI，查重修改的成果会被 AI 降重破坏。

## 复检策略

1. 改完后先自查：对照 AI 特征词黑名单 grep 全文
2. 送检前确保全文风格一致
3. 第一次送检后，针对仍然标红的段落二次修改
4. 同一文本二次检测结果可能浮动 ±5%，属于正常范围
